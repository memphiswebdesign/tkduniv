This is an **example custom page**. It was created to confirm the new "Add a Page" feature works end to end. You can safely delete it from the CMS once you've tested things.

## What you can do here

- Create a brand-new page from the **Custom Pages** collection in the CMS
- Give it a title, an optional subtitle, and a meta description for SEO
- Write the body content with the rich-text editor

The page is automatically published at its own web address based on the title — this one lives at `/example-page/`.

> Heads up: a new page is **not** automatically added to the site's navigation menu. A developer links it into the menu when you're ready for visitors to find it.
